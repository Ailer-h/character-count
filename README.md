This project is a simple, de-bloated and fast way to type out something and have statistics about what you're typing.
It was put together in a single morning because I was tired of using ones that were full of ads.

Feel free to use as it fits your workflow. You can access it [here](https://ailer-h.github.io/character-count/).

## Features

-   Live character, word and paragraph counting;
-   Autosave to local storage, with a toggle and a "last saved" label;
-   Markdown preview - hit "Preview MD" to open a split view with the rendered markdown side by side with the editor;
-   Markdown editor niceties - pressing Enter inside a list or blockquote continues it automatically (numbered lists increment on their own), and pressing Enter on an empty item exits the list;
-   Supports headers, bold/italic/strikethrough, links, inline code, code blocks, blockquotes, ordered/unordered lists, horizontal rules and tables (with column alignment).

### What's in the horizon?
I plan on adding features like stating a limit notification so it notifies you when you go over it and support for uploading/downloading files but this project is not a priority so it will probably take a long time.
